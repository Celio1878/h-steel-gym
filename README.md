## About The Project

H-Steel Gym is a modern web application designed for gyms and fitness centers to showcase their facilities, services, and workout programs. Built with the latest technologies like React Router 7 and Tailwind CSS 4, it provides a seamless, responsive, and high-performance experience for users.

Key features:
* **Dynamic Gallery**: High-quality showcase of gym machines and facilities.
* **Service Listing**: Detailed personal training, nutrition planning, and group class options.
* **Workout Programs**: Various training types like Hypertrophy, HIIT, and Powerlifting.
* **Multi-language Support**: Automatic localization for English and Portuguese (Brazil/Portugal).
* **Dark Mode**: Fully themed experience with light/dark mode switching.

<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/celio1878/h-steel-gym.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the development server
   ```sh
   npm run dev
   ```

## Usage

H-Steel Gym can be used as a template for fitness centers looking to establish a professional online presence. The application automatically detects the user's location to serve content in their preferred language (Portuguese for Brazil/Portugal, English otherwise).

_For more examples, please refer to the [Documentation](https://github.com/celio1878/h-steel-gym)_
